Codigo Ascensor
===============

Autor: Jose Manuel Rodriguez Montes

Codigo Java que simula un ascensor, asi como sus ocupantes, hay dos modos de funcionamiento:
	- Aleatorio: El tiempo de los pasajeros en las plantas y las plantas destinos son aleatorias.
	- Determinista: Se mira un fichero donde se encuentran los tiempos y las plantas de los usuarios.

Elevator Code
=============

Author: Jose Manuel Rodriguez Montes

Java code simulated a lift and his users, there are two mods:
	- Random: Time and floors are random selected by program.
	- Determinist: It looks a file where is located times and floors of users.
